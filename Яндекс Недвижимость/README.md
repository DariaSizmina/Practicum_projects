# Описание проекта - Яндекс Недвижимость


## Данные
В наличии были следующие данные о заемщиках: 
- Количество детей
- Семейное положение
- Уровень дохода
- Цель кредита

## Задача

Произвести анализ факторов, влияющих на возврат кредита в срок, платежеспособность заемщиков.
